# 🚀 AI-Powered Smart Document & Vehicle OCR System

An advanced Optical Character Recognition (OCR) application that extracts, analyzes, and processes text from images using AI.

---

## 📌 Features

✅ Extract text from images (documents, bills, number plates)  
✅ Bounding box visualization with confidence score  
✅ Structured data extraction (Date, Amount, Vehicle Number)  
✅ Multi-language support (English, Hindi, Marathi)  
✅ Search text within extracted results  
✅ Download extracted text as file  
✅ Text-to-Speech (optional)  
✅ Text Translation (Hindi support)  
✅ Clean and interactive UI  

---

## 🧠 Technologies Used

- Python  
- EasyOCR  
- OpenCV  
- Streamlit  
- NumPy  
- Pillow  
- Pyttsx3 (Text-to-Speech)  
- Deep-Translator  

---

## 🏗️ Project Architecture

1. Image Upload  
2. Image Preprocessing (Grayscale, Thresholding)  
3. Text Detection using EasyOCR  
4. Text Extraction & Confidence Scoring  
5. Structured Data Processing  
6. Output Display + Download
